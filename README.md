# SoonFramework
基于.Net Framework 4.6的开发框架
## 组件

### SoonFramework.Core
核心，定义一些常用的异常类。
### SoonFramework.Web
Asp.Net MVC5 工具模块
### SoonFramework.Security
身份验证模块
### SoonFramework.Security.Operator
身份验证配套的管理业务逻辑
### SoonFramework.Data
数据库访问模块
### SoonFramework.Weather
天气数据
### SoonFramework.WebTheme
Web系统主题支持
### SoonFramework.Personalize
个性化
